# clustered-backend
As the name specifies, this is backend for clustered project. This repo is currently being hosted as a server on Vercel

## How it's made
The packages used in backend are:
<ul>
  <li>ExpressJs</li>
  <li>Mongoose</li>
  <li>CORS</li>
  <li>Axios</li>
  <li>Body-parser</li>
</ul>
